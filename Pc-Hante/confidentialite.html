Politique de confidentialite

Derniere mise a jour : juin 2026

Donnees collectees

La Majorite collecte uniquement les informations necessaires au fonctionnement du jeu :

  • pseudonyme du joueur ;
  • avatar choisi ;
  • scores, statistiques de jeu, streaks et succes ;
  • reponses fournies aux questions ;
  • messages envoyes dans le chat public ;
  • identifiant joueur stocke dans le navigateur pour reserver le pseudo ;
  • donnees de navigation anonymisees via Google Analytics.

Utilisation des donnees

Ces donnees sont utilisees uniquement pour :

  • afficher les classements ;
  • calculer les statistiques du jeu ;
  • afficher les profils, succes et messages du chat ;
  • ameliorer l'experience utilisateur ;
  • eviter que plusieurs joueurs utilisent le meme pseudo ;
  • mesurer la frequentation du site et comprendre les pages les plus consultees.

Partage des donnees

Les donnees ne sont ni revendues ni transmises a des tiers a des fins commerciales.

Le site utilise des prestataires techniques necessaires a son fonctionnement, notamment Netlify pour l'hebergement, Supabase pour la base de donnees et Google Analytics pour les statistiques de visite.

Conservation et suppression

Les scores, votes, pseudos et messages peuvent etre conserves tant que le jeu est en ligne afin de maintenir les classements et statistiques.

Vous pouvez demander la suppression ou la correction de vos donnees en contactant :

baptiste.benard.bachelard@gmail.com

Vos droits

Conformement au RGPD, vous pouvez demander l'acces, la rectification ou la suppression de vos donnees personnelles.
